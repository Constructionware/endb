    • Third-Party-Adapters

    You can integrate and use third-party adapters or build your own.

    const myAdapter = require('./my-adapter');
const endb = new Endb({ store: myAdapter });

    Any module/package that follows the Map API can be integrated.

    new Endb({ store: new Map() });

    For example, quick-lru is an unrelated module that implements the Map API. However, extension methods (all and find) may not work.

    const Endb = require('endb');
const QuickLRU = require('quick-lru');
const lru = new QuickLRU({ maxSize: 1000 });
const endb = new Endb({ store: lru });

    List of third-party adapters supported by Endb:

    • quick-lru - Simple "Least Recently Used" (LRU) cache
